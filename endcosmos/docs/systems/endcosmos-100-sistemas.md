# EndCosmos: 100 Sistemas Definitivos del Mundo

Documento maestro para diseño e implementación ordenada del worldbuilding MMORPG de EndCosmos.

## Núcleo del universo

- **Aurelia Prime**: capital del mundo
- **Draco**: guerra y conquista
- **Llunaris**: conocimiento y magia
- **Uruk Reborn**: origen de civilización

## Consejo de implementación

No implementar todo a la vez. El orden recomendado inicial es:

1. Sistema de mapa global conectado
2. Sistema de clima dinámico
3. Sistema de bosses globales
4. Sistema de eventos globales persistentes
5. Sistema de exploración secreta
6. Sistema de control territorial

Con estos 6 sistemas bien construidos, el mundo ya se percibe vivo, escalable y persistente.
